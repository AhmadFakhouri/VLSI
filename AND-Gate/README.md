# CMOS AND Gate Transient Analysis

This project involves the implementation and timing verification of a CMOS 3-bit AND gate. The focus was on measuring high-speed transient performance and propagation delays.

## Performance Results

* **Rising Edge Delay**: Measured at 318.9 ps.
* **Falling Edge Delay**: Measured at 325.3 ps.

## Repository Contents

* **Schematic_Modules**: Full-custom AND gate schematic.
* **Transient-Analysis**: Waveform screenshots showing thetiming.
